记本博客的诞生

  • hexo
  • 经验

有搭博客的想法很久了，拜万恶的拖延症所赐，一直只有想法没有付诸行动，此次趁着自己刚好买了台新的 15吋 RMBP ，兴致又起来了。

安装过程没什么好赘述的，毕竟网上成熟的教程很多，这边写写自己在搭建本博客的过程中碰到过的问题：

· 安装 Git

· 安装 node.js (安装的方式有很多种，可选pkg安装包安装，但是不知道为什么我用pkg安装一直失败，后改用homebrew安装成功了)

· 安装 hexo 也不难，$ npm install -g hexo即可，如果失败一般是权限问题，改用 $ sudo npm install -g hexo就可以了。

· 配置 _config.yml 时，要注意，文件中每个选项的冒号:后面都必须空一格，否则会报错。

· hexo 3.0 之后，_config.yml 的 deploy 设置，type要设置成git。

· hexo 3.0 之后，hexo 的作者 tommy 为了使博客生成静态文件和部署的速度加快，将插件都剥离出来了，所以部分插件要独立安装。（如：hexo-deployer-git）

·
·
·

· 搭建成功，我犯了一个非常非常low的错误，在 _config.yml 中配置 deploy 那一块时，我竟然手抖将 repository 写成了 respository ，就接这点纠结了大半天。。杀了我吧

最后，搭建 hexo 博客就跟写代码一样，过程也非常有趣 ^_^。
